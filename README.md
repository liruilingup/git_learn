# git_learn
方式一：添加远程仓库
1.建立本地仓库
在文件夹下运行：git init
2.创建远程github仓库
直接在github网站上面创建
3.将本地仓库和远程仓库关联
本地运行：git remote add origin git@github.com:liruilingup/git_learn.git
本地内容推送：git push -u origin master
本地commit后：git push origin master

方式二：从远程库克隆
一、创建远程github仓库
直接在github网站上面创建
二、克隆一个本地库
在本地文件夹运行：
git clone git@github.com:liruilingup/git_learn.git
三、进入git_learn文件夹
